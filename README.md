# SEFFER — Система автоматического переименования фото (Double YOLO)

Интеллектуальное веб-приложение для пакетной обработки фотографий с распознаванием серийных номеров. Решение упаковано в Docker-контейнер для мгновенного запуска на любой системе.

## ✨ Ключевые возможности
- **Нейросетевой пайплайн**: Использование двух моделей YOLO (OBB + OCR) для точного распознавания меток.
- **Docker-ready**: Готовый образ, не требующий установки Python, библиотек или CUDA.
- **Пакетная обработка**: Удобная валидация сотен фото с сохранением истории в Excel.

## 🛠 Технологии
- Python, Streamlit, PyTorch (YOLO), OpenCV, Docker.

## 🚀 Запуск для пользователя (Docker)

Вам не нужно собирать проект или устанавливать зависимости. Просто выполните команду в терминале, указав папку на вашем компьютере, куда будут сохраняться результаты:

```bash
docker run -p 8501:8501 -v "C:\ВАША_ПАПКА_ДЛЯ_РЕЗУЛЬТАТОВ:/app/output" ghcr.io/ОльгаМ-ДС/seffer_auto_naming:latest

После запуска приложение будет доступно в браузере по адресу: http://localhost:8501
